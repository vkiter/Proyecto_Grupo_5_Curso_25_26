# MusiKeasy — *"Escucha la diferencia"*
 
MusiKeasy es una plataforma web de música orientada a la **promoción de artistas emergentes**, con un enfoque bidireccional entre artista y oyente. A diferencia de las grandes plataformas tradicionales (Spotify, Apple Music), MusiKeasy ofrece a cualquier músico un espacio propio donde publicar su obra, acceder a estadísticas reales de escucha y establecer una relación directa con su audiencia, sin intermediarios.
 
Los usuarios oyentes pueden valorar canciones, comentar publicaciones, apoyar económicamente a los artistas mediante donaciones y descubrir nuevo talento emergente, construyendo una comunidad activa alrededor del contenido musical.
